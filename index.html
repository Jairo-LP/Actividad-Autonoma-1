¿Qué es el DOM?

El Document Object Model (DOM), conocido en español como Modelo de Objetos del Documento, Es el objeto basico del lado de cliente que se le proporcionana a aceeso a la pagina web, el cual proporciona propiedades y metodos en el acceso a los obejtos de ventanas del navegador

  • Document.body: Devuelve el elemento document
  • Document.title: Devuelve el elemento titulo de pagina
  • Document.images: es un array que representa una imagen por pagina
  • Document.bgcolor: Define color de fondo, definiendo el color en ingles

    • En términos simples, el DOM convierte el código HTML estático en una estructura de objetos que puede ser modificada en tiempo real por programadores mediante código HTML, lo que permite crear experiencias interactivas para el usuario.

    Características principales

    Estructura de árbol

    El DOM organiza los elementos del documento en una estructura jerárquica en forma de árbol. El nodo raíz corresponde al elemento <html>, del cual se desprenden todos los demás elementos como hijos y subnodos (Escarcena, s.f.).

    Nodos

    Cada elemento, atributo y texto dentro del documento representa un nodo. El DOM define distintos tipos de nodos: nodos de elemento, nodos de texto, nodos de atributo, entre otros. Todos ellos forman parte del árbol del documento y pueden ser manipulados independientemente.

    Independencia del lenguaje

    Aunque el DOM se asocia frecuentemente con JavaScript, es una interfaz que puede ser utilizada con cualquier lenguaje de programación que implemente sus especificaciones, como Python, Java o PHP (Programación de Páginas Web con JavaScript y PHP).

    Manipulación dinámica

    El DOM permite agregar, eliminar o modificar elementos y contenido de la página sin necesidad de recargar el documento completo, lo que es la base de las aplicaciones web modernas e interactivas.

    Objetivos y utilidad en el desarrollo web

    • Acceso al contenido: El DOM permite leer y modificar el contenido de la página, como el texto dentro de un párrafo o el valor de un campo de formulario.
    • Modificación de la estructura: Permite añadir o eliminar elementos HTML de forma dinámica según las necesidades de la aplicación.
    • Manipulación de estilos: Mediante el DOM es posible cambiar las clases CSS y estilos en línea de los elementos para alterar su apariencia en tiempo de ejecución.
    • Manejo de eventos: El DOM establece un modelo de eventos que responde a las acciones del usuario, como clics, pulsaciones de teclado y movimientos del ratón.
    • Interactividad: Es la base para crear interfaces dinámicas y experiencias de usuario ricas sin depender únicamente del servidor.
